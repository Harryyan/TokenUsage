你现在是一个资深的 macOS / Swift 工程师，请帮我设计一个原生 macOS Menu Bar App，用来展示我当前机器上的 Claude Code 使用情况。

这个项目的数据来源是 **ccusage**。请围绕 ccusage 作为唯一核心数据来源来设计方案和代码骨架。

# 一、产品目标

开发一个 **macOS Menu Bar App（菜单栏应用）**，用于实时展示 Claude Code 的 token 使用情况和对应费用。

这个 App 的核心目标是：

1. 在 **Menu Bar** 上默认实时显示当前 token 使用量
2. 显示时使用简洁缩写格式，便于在菜单栏中阅读
3. 点击 Menu Bar 图标后，可以看到更详细的 usage 信息
4. 支持按 **Today / This Week / This Month** 展示 token 使用量和对应价格
5. 展示累计总费用
6. 数据每隔 **5 分钟自动刷新一次**
7. UI 风格简洁、专业、原生、适合开发者工具

---

# 二、核心交互需求

## 1. Menu Bar 默认显示内容

Menu Bar 默认实时显示当前 token 使用量，要求：

- 显示的是当前 usage 的核心数字
- 数字使用缩写格式，适合菜单栏有限空间
- 例如：
  - 1200 -> 1.2K
  - 1250000 -> 1.25M
  - 10000000 -> 10M
  - 23500000 -> 23.5M

要求：
- 缩写逻辑清晰、一致、易读
- 默认显示简洁值
- 用户点击展开后，可以看到**精确完整数值**

请你帮我设计一个适合菜单栏场景的显示方案，包括：
- 只显示 token
- token + 价格
- 图标 + 数值
- 数值长度过长时如何处理
- 菜单栏空间受限时的优雅降级策略

## 2. 点击展开后的详情面板

点击 Menu Bar 后，弹出一个详情面板（Popover 或 Menu Bar Panel），展示完整 usage 信息。

详情面板中需要包含：

### A. 当前实时信息
- 当前 token 使用量（完整精确数字）
- 当前对应费用
- 最近一次刷新时间
- 当前刷新状态（成功 / 刷新中 / 失败）
- 如果适合，显示一个小型趋势摘要

### B. Today（今日）
展示：
- 今日 token 使用量
- 今日对应费用
- 今日精确数值
- 今日趋势或分段情况（如适合）

### C. This Week（本周）
展示：
- 本周 token 使用量
- 本周对应费用
- 本周日均 token
- 本周日均费用
- 本周总费用

### D. This Month（本月）
展示：
- 本月 token 使用量
- 本月对应费用
- 本月日均 / 周均 usage
- 本月总费用

### E. Overall / Total
展示：
- 累计 token 使用量
- 累计总费用

# 三、数据与展示要求

请将数据展示拆成以下几个层级：

## 1. 菜单栏层（最简展示）
这是用户最常看到的一层，要求：
- 信息非常简洁
- 默认显示当前 token usage 的缩写值
- 可选显示费用缩写
- 刷新后自动更新

## 2. 弹出层（详细展示）
这是用户点击 Menu Bar 之后的详细信息层，要求：
- 展示完整数字，不做模糊缩写
- 明确展示 token 和 price 的对应关系
- 区分 Today / Week / Month / Total
- 信息层级清晰，不要太拥挤

## 3. 数字格式化要求
请设计统一格式化策略：

### Token 缩写规则
- K = 千
- M = 百万
- B = 十亿
- 保留合适小数位
- 菜单栏里尽量短
- 详情页里显示精确值

例如：
- 15320 -> 15.3K
- 1250000 -> 1.25M
- 10000000 -> 10M
- 23800000 -> 23.8M

### 费用格式
- 使用货币格式显示
- 菜单栏中可简写
- 展开后显示精确金额
- 例如：
  - $12.30
  - $1.2K（如果金额非常大）
- 请考虑不同精度展示场景

# 四、刷新机制

请按下面要求设计实时更新方案：

- App 启动后立即加载
- 每隔 **5 分钟自动刷新**
- 支持手动刷新
- 刷新失败时保留上一次成功结果
- UI 要能显示：
  - idle
  - loading
  - refreshing
  - success
  - failed

对于菜单栏应用，请特别考虑：
- 如何避免过于频繁刷新导致性能浪费
- 如何避免多个刷新任务并发
- 如何在 App 长时间运行时保持稳定
- 如何在 Menu Bar 场景下优雅展示刷新状态
